### 🐱 What is AniNeko?
**AniNeko** is a desktop project built for fun. It puts an interactive anime cat (starting with a **Miku** version) on your screen that treats your mouse cursor like a toy, chasing it around while you work or browse.

### ✨ Highlights
* **Active Tracking:** Calculates the distance to your cursor to trigger "chase" or "watch" animation cycles.
* **Current Cats:** Using **Sora.ai** for cat pixelated art to keep animations smooth and expressive.
* **Lightweight:** Built with Java to stay light on system resources.

### 🛠️ How to Run
1. **Clone the repo:** `git clone https://github.com/YanirValdman/AniNeko.git`
2. **Launch:** Open the project in your IDE or run the JAR file directly.

### 🎨 Asset Note
I'm doing this totally for fun and using AI for the cat art because I'm no Picasso. If you want to contribute hand-drawn sprites or new characters, feel free to open a Pull Request!
